# 产品经理知识库

欢迎来到产品经理知识库！这里为你提供系统化的产品经理学习与成长指南。

## 核心模块

### [产品基础](/pm-basics/intro)

产品经理的基础知识体系，包括：
- 角色定义与职责
- 核心技能培养
- 产品方法论
- 工具使用指南
- 文档写作规范

### [电商产品](/ecommerce/intro)

电商领域产品经理必备知识：
- 电商业务模式
- 用户研究方法
- 运营策略
- 数据分析
- 经典案例研究

### [供应链产品](/supply-chain/intro)

供应链领域产品经理专业知识：
- 供应链领域知识
- 业务流程设计
- 系统架构
- 优化方法论
- 实战案例分析

### [产品求职](/job-hunting/intro)

产品经理求职与发展指南：
- 简历优化指南
- 面试技巧
- 案例分析准备
- 职业发展规划
- 求职渠道推荐

## 使用指南

1. 新手建议从[产品基础](/pm-basics/intro)开始系统学习
2. 对特定领域感兴趣的同学可以直接进入相应模块
3. 求职准备的同学可以参考[产品求职](/job-hunting/intro)模块
4. 文档持续更新中，欢迎提出建议和反馈
