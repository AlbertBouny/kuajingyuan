# 禅道：国产开源项目管理系统

## 一、工具简介

禅道是一款国产的开源项目管理软件，集产品管理、项目管理、测试管理于一体。

```
基础功能：

- 项目管理
- 测试管理
- 文档管理
- 事务管理

特色功能：
```
