## 学习建议

1. 建议按照学习路径顺序学习，循序渐进
2. 每个模块都包含理论和实践部分，建议：
   - 先理解基础概念和方法论
   - 再通过实践案例加深理解
   - 最后结合工具进行实战演练

3. 重点关注：
   - 角色认知：明确产品经理的职责和定位
   - 基本技能：掌握核心工作方法
   - 工具使用：熟练运用常用工具
   - 文档写作：提升文档输出能力

4. 持续学习：
   - 关注行业动态和新趋势
   - 积累实战经验和案例
   - 培养产品思维和视野

::: tip 学习路径
👉 新手建议按照：角色认知 → 基本技能 → 方法论 → 工具使用 → 文档写作 的顺序学习
:::

::: warning 注意事项
- 理论学习要结合实践
- 多参考优秀案例
- 注重实战演练
- 持续积累经验
:::
